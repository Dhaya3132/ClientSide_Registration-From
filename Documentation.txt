Removing the data :

I have adding eventlistner to table body, by which i can know which element has been clicked by using event targeting the element .
If the clicking element contains any icons class i just wanna to remove their parent element.

CODE :

TargetList.addEventListener('click', (e)=>{
    if(e.target.classList.contains('icons'))
    {
      e.target.parentElement.parentElement.parentElement.remove();
    }
});
